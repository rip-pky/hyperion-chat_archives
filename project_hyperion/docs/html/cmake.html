CMake do Project Hyperion

Este diretório contém o padrão de build e documentação para facilitar a configuração em código aberto.

Arquivos

  • docs/cmake/README.md: como usar os presets e rodar o CMake.
  • docs/cmake/strings.md: explicação do significado das strings e variáveis do CMake.
  • CMakeLists.txt: script principal de build na raiz.

Como usar

cd C:\Users\PC\Desktop\project_hyperion
cmake --preset debug
cmake --build --preset debug
