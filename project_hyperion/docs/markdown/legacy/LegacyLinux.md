# Versão Legacy no Linux

O `legacy_web/` funciona como versão legada em PHP em Linux, Mac e Windows.

## Como executar no Linux

1. Instale PHP:
   ```bash
   sudo apt install php
   ```
2. Abra a pasta `legacy_web/`:
   ```bash
   cd legacy_web
   php -S localhost:8000
   ```
3. Abra `http://localhost:8000/index.php`.

## Nome sugerido

- `Ligax` para a versão Linux principal.
- `Hyperion` para a versão Mac e Windows.

## Observações

- A versão legacy usa HTML 4.01 Transitional e CSS simples.
- `dev.php` permite testar o modo de desenvolvimento e debug em qualquer sistema.
