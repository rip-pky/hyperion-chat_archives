# Legacy Web PHP

Esta pasta contém a versão web legada do Project Hyperion, projetada para navegadores antigos.

## Arquivos importantes

- `index.php`: interface de chat legada e compatível com HTML 4.01.
- `dev.php`: versão de desenvolvimento com painel de debug.
- `download_css.php`: permite baixar o CSS.
- `css/style.css`: estilo principal para a versão legacy.
- `css/dev.css`: estilo para a versão de desenvolvimento.

## Como testar

1. Execute um servidor PHP local na pasta `legacy_web/`.
2. Acesse `http://localhost:8000/index.php`.
3. Use `dev.php` para testar o modo dev.
4. Baixe o CSS em `download_css.php`.
