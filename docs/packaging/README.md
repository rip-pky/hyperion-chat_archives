# Packaging do Project Hyperion

Esta pasta contém scripts e instruções para criar builds, instaladores e pacotes para Windows, Linux e macOS.

## Estrutura

- `packaging/windows/`: build e instalador para Windows.
- `packaging/linux/`: empacotamento para Linux com nome `Ligax`.
- `packaging/macos/`: empacotamento para macOS.

## Objetivo

Fornecer um caminho claro para gerar:
- instaladores Windows (`.exe` ou `.msi`)
- pacotes Linux (`.tar.gz`, AppImage ou `Ligax`)
- pacotes macOS (`.dmg`/`.pkg`)

Use os arquivos dentro de cada subpasta para executar a criação de pacotes.
