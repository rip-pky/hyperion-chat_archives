# Deploy e Fork para Project Hyperion

Esta pasta contém instruções e scripts para empacotar a versão web legacy, documentação e extensões.

## Como usar

- `package_mod.ps1`: empacota os arquivos `legacy_web/` e `docs/markdown/` em um ZIP pronto para compartilhamento.
- `package_mod.bat`: mesma função para Windows sem PowerShell.

## Recomendação

Use este pacote se quiser criar um fork, compartilhar um mod ou oferecer uma versão customizada com documentação sólida.
