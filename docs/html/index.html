Project Hyperion

Chat gamer com foco em privacidade extrema, segurança ativa e personalização de perfil com temas e status.

Visão geral

O projeto reúne frontend C++ com Qt6 e backend Python para criar um ambiente de chat seguro e com baixa latência, além de recursos de perfil altamente customizáveis.

Recursos principais

  • Perfil gamer personalizado: nickname, status, disponibilidade e temas.
  • Buffer de mídia em RAM: uploads mantidos apenas em memória e limpos imediatamente.
  • Conexão TCP/TLS segura: dados trafegam criptografados e com handshake inicial.
  • Watchdog Python: validação HWID, rate limiting e blacklist de conteúdo.

Arquitetura do projeto

  • src/: código C++ do cliente e UI.
  • backend/: watchdog Python e listas de segurança.
  • cmake/: presets e documentação de build CMake.
  • docs/html/: documentação detalhada do projeto.
  • docs/webclip/: versão compacta para previews e quick reference.

Como compilar

Recomenda-se usar Qt6 e CMake no Windows. Execute os comandos abaixo no PowerShell:

cd C:\Users\PC\Desktop\project_hyperion
mkdir build
cmake -S . -B build
cmake --build build --config Debug

Execução

build\Debug\ProjectHyperion.exe

Também é possível usar os scripts:

  • build_and_run.ps1 (PowerShell)
  • build_and_run.bat (CMD)
  • run_project.bat para executar o binário já compilado

Segurança

  • HWID autenticado com allowlist local.
  • Rate limiting e bloqueio de IP para tentativas de ataque.
  • Blacklist de hashes SHA-256 para mídia proibida.
  • SSL/TLS no backend para proteção de transporte.
