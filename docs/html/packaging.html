Packaging do Project Hyperion

Guia de empacotamento para Windows, Linux e macOS, com versões legacy e instaladores.

Plataformas suportadas

  • Windows: instalador NSIS (`.exe`).
  • Linux: pacote `Ligax` em tarball.
  • macOS: `.pkg` e `.dmg`.
  • Legacy: versão PHP para navegadores antigos.

Como usar

cd C:\Users\PC\Desktop\project_hyperion
cmake --preset debug
cmake --build --preset debug

Depois, siga o pacote correspondente:

  • Windows: `packaging/windows/build_installer.ps1`
  • Linux: `packaging/linux/build_ligax_package.sh`
  • macOS: `packaging/macos/build_ligax_dmg.sh`
