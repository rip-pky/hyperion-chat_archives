## 🛰️ O Protocolo Definitivo
O **Hyperion** é um motor de chat descentralizado de alta performance. Esqueça infraestruturas centralizadas. Aqui, o código em **C++** puro gerencia os dados, enquanto você dita as regras com **HTML, CSS, JS e TS**.

Se você tem este repositório, você tem o poder de criar uma rede de comunicação imune a censura e controle corporativo.

---

## 🛠️ Arquitetura do Sistema

### 📂 Estrutura de Pastas
* `src/` -> O coração do monstro. Todo o Core em **C++**.
* `docs/` -> Manuais de implementação, lógica do **Watch Dogs** e guia de segurança.
* `web/` -> Onde a mágica acontece. Templates base em HTML/CSS/JS para o seu Front-end.
* `scripts/` -> Automações para deploy e configuração rápida.

### ⚡ Build de Elite
O projeto utiliza **Ninja** para uma compilação absurdamente rápida, ideal para quem desenvolve em máquinas leves ou TV Boxes.

```bash
# Clone o Império
git clone [https://github.com/rip-pky/hyperion-chat_doc.git](https://github.com/rip-pky/hyperion-chat_doc.git)
cd hyperion-chat_doc

# Compile o Motor
cmake -G Ninja -B build
cmake --build build
